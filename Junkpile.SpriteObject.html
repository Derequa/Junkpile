Class: SpriteObject

Junkpile.SpriteObject

This class wraps up a PIXI Sprite as an object in the game-object model.

Constructor

new SpriteObject(engine, name, parent, sprite, layer)

This contructs the component attached to the given parent and sprite.
Parameters:
Name Type Description
engine Junkpile.Engine The engine instance this sprite is apart of.
name string A unique name string to mark this object. (optional)
parent Junkpile.model.EngineComponent The parent component for this component.
sprite PIXI.Sprite The PIXI sprite object to wrap in this component.
layer number The layer in the engine this sprite is drawn on.
