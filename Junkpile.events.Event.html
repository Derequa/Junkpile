Class: Event

Junkpile.events.Event

This class provides a base event representation for events to extend and implement in the engine's event system.

Constructor

(abstract) new Event(eventdata, typeID)

This defines the constructor for a generic event object.
Parameters:
Name Type Description
eventdata object The obejct representing specific data for the implementing event. This can be a recieced event object or position data etc..
typeID string A string representing the type of implementing event. This should also be defined as a constant for reference in the implementing class file.
