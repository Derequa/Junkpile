Class: KeyEvent

Junkpile.events.KeyEvent

This class wraps a key event from the document to a typed event to be used in the engine's event system.

Constructor

new KeyEvent(documentEvent, keyDown)

This contructs a KeyEvent for the engine from the document event and the given key state.
Parameters:
Name Type Description
documentEvent KeyboardEvent The event recieved from the document.
keyDown boolean A boolean for if the key was pressed. True if pressed, false if released.
