Class: Camera

Junkpile.components.Camera

This class implements a camera that will move the engine's view box with a user.

Constructor

new Camera(engine, name, parent)

This contructs a new Camera for a given user, using a givne engine instance.
Parameters:
Name Type Description
engine Junkpile.Engine The engine this is in. The renderer, view, and world will be used from this instance.
name string The unique name for this component, for easy finding.
