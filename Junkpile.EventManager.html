Class: EventManager

Junkpile.EventManager

This class implements a dynamic typed event management system. This allows for any type of event to be sent to the system, classes and components to listen for any given type of events, and provide any fuction to be notified when that event occurs.

addListenerFor(typeID, handlerObj, handlerFunc)

This adds a listener for a given type of event.
This:
  • {Junkpile.EventManager}
Parameters:
Name Type Description
typeID string The event type to listen for. Each event class maintains a constant string that will be set for the type field on construction. These can be used to register as a listener.
handlerObj object The instance of the object to call the handler function on.
handlerFunc function The call-back function to handle the given type of event. This function will be called using the form: function(event, handlerObj) Where the event field is the event object and the handler object is the same handler object given here. Design your functions to match and accept this standard.
